Nuestro Compromiso

En {{ company_name }}, estamos comprometidos a garantizar que nuestro sitio web sea accesible para todas las personas, incluidas aquellas con discapacidades.

Estándares de Accesibilidad

Nos esforzamos por cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1 nivel AA, que son reconocidas internacionalmente como estándar para la accesibilidad web.

Características de Accesibilidad

Nuestro sitio web incluye las siguientes características de accesibilidad:

  • Navegación por teclado: Todas las funciones son accesibles mediante el teclado
  • Texto alternativo: Todas las imágenes tienen descripciones alternativas
  • Contraste de color: Utilizamos combinaciones de colores con suficiente contraste
  • Estructura clara: Uso de encabezados y listas para organizar el contenido
  • Texto redimensionable: El texto se puede ampliar hasta un 200% sin perder funcionalidad
  • Formularios accesibles: Etiquetas claras y mensajes de error descriptivos

Tecnologías Compatibles

Este sitio web es compatible con:

  • Lectores de pantalla como NVDA, JAWS y VoiceOver
  • Navegadores web actuales (Chrome, Firefox, Safari, Edge)
  • Tecnologías de asistencia para navegación por teclado
  • Software de reconocimiento de voz

Limitaciones Conocidas

Aunque nos esforzamos por la máxima accesibilidad, pueden existir algunas limitaciones:

  • Algunos contenidos de terceros pueden no cumplir completamente con los estándares de accesibilidad
  • Estamos trabajando continuamente para mejorar la accesibilidad de todas las funciones

Mejoras Continuas

Estamos comprometidos con la mejora continua de la accesibilidad de nuestro sitio web. Nuestros esfuerzos incluyen:

  • Auditorías regulares de accesibilidad
  • Pruebas con usuarios con discapacidades
  • Capacitación continua de nuestro equipo de desarrollo
  • Actualización regular de nuestras prácticas de accesibilidad
